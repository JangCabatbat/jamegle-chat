Jamegle - Chat with People of Similar Interests
Jamegle is a simple chat application that allows users to connect with strangers based on shared interests. This project is built using HTML, CSS, and JavaScript and is designed to be an interactive and user-friendly chat experience.

Features
Start and Stop Chatting: Initiate or terminate the chat session at your convenience.
Send Messages: Engage in text-based conversations with a stranger.
Find Match Based on Interests: Connect with someone who shares similar interests.
Simulated Typing Indicator: Get a sense of when the stranger is typing.
